# SELECT
```
SQL關鍵字沒有英文字母大小寫之分，而且可以寫成多行或一行
SELECT   欄位名稱
FROM      資料表名稱
[WHERE  搜尋子句]
[ORDER BY  排序子句 {ASC|DESC}]
```
```
INSERT INTO 資料表名稱 (欄位1, 欄位2, 欄位3…) Values (資料1, 資料2, 資料3…)
```
